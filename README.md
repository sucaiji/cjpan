# CJPAN私有云服务

## 概述
CJPAN是类似于owncloud的私有云平台，可以通过简单的部署建立属于自己的私人网盘
## 功能
- 上传下载文件
- 视频在线观看
- 图片在线预览
- 文档在线预览(未完成)
- 文件分享(未完成)
- 多平台客户端(未完成)

#### 使用方法
下载该文件之后打开所在的目录，然后打开该目录的命令行窗口，然后执行```java -jar xxx.jar```（xxx指的是该文件的名称，一般是叫cjpan-0.1.2这样类似的，操作时以实际名称为准），然后在浏览器中访问localhost，如果显示出来网盘界面，那么恭喜你搭建成功了。
